# Feature: Game Configuration Setup

## Overview
This document provides a design summary for the game configuration screen. This feature allows players to customize their game experience by choosing a role and setting the game duration.

## Key Design Goals
- **Clarity**: Clearly present the available roles and their descriptions.
- **Simplicity**: Make the configuration process quick and intuitive.
- **Feedback**: Provide immediate feedback on user selections.
