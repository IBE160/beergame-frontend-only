# Interactions: Interactive Tutorial

- **Progression**: The user clicks "Next" to move between steps, or the tutorial can advance automatically when the user performs the correct action.
- **Tooltips**: The tooltips appear and disappear with a smooth fade animation.
- **Skip**: The user can skip the tutorial at any time by clicking the "Skip" button, which will then be confirmed with a modal.
