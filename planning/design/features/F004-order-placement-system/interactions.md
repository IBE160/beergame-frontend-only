# Interactions: Order Placement

## 1. Input Field

- **Interaction**: Real-time validation as the user types.
- **Feedback**: If an invalid character is entered, it is immediately rejected or an error message is shown.

## 2. Confirmation Modal

- **Trigger**: User submits an order over a certain threshold (e.g., 100).
- **Animation**: The modal fades in over the dashboard.
- **Interaction**: The user must click "Confirm" or "Cancel". The modal can be dismissed by pressing the Escape key.
